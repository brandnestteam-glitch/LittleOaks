# LittleOaks 🌳

Welcome to the **LittleOaks** repository! This is a modern, interactive web application built with Next.js and React.

## 🚀 Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **UI Library:** [React](https://react.dev/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/), [GSAP](https://gsap.com/), and [Vanilla Tilt](https://micku7zu.github.io/vanilla-tilt.js/)
- **Components:** [Swiper](https://swiperjs.com/) for carousels, [@phosphor-icons/react](https://phosphoricons.com/) for icons
- **Utilities:** `pdf-lib` for PDF handling, `resend` for emails

## 🛠️ Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/brandnestteam-glitch/LittleOaks.git
   cd LittleOaks
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up Environment Variables:**
   - There is a `.env.local` file required for this project.
   - Ensure you have the necessary environment variables set up (like API keys, Resend keys, etc.).

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the website in action.

## 🌐 SEO & Deployment

- **Sitemap & Robots:** The project includes `sitemap.js` and `robots.js` inside `src/app` for Google Search Console & SEO readiness. 
  - **Action Required:** Make sure to update the domain placeholder in these files with your live domain before deploying.
- **Deployment:** We recommend deploying this Next.js project on [Vercel](https://vercel.com/) for the best performance.
